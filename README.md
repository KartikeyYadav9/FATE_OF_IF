FATE_OF_IF

a python terminal based horror game which is choice driven

How to play (for windows)

1. Click on the website link ( which is https://github.com/KartikeyYadav9/FATE_OF_IF/releases/download/v1.0.0/fate_of_if.zip)
2. Unzip the downloaded file
3. You will have the .exe file
4. Run it

NOTE - Use of headphones is recommended for a better feel

Feature

* terminal based interface
* sound effects for better feel
